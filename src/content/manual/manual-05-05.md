
## 5.5 전략 가져오기 / 내보내기

### 개요

**가져오기 / 내보내기** 기능은 매매 전략을 **JSON 파일 형식**으로 저장하고 다시 불러올 수 있는 기능입니다.

이 기능을 활용하면 다음과 같은 작업이 가능합니다.

*   잘 동작하는 전략을 **백업**하여 보관
*   다른 PC로 전략을 **이전**
*   동일 전략을 **복사**하여 변형 전략 생성
*   전략 **공유**

내보내기 / 가져오기 버튼 위치

**자동매매 만들기** 화면의 전략 목록에서 오른쪽 클릭(컨텍스트 메뉴) 또는 전략 목록 상단 메뉴에서 접근합니다.

## 5.5.1 전략 내보내기 (Export)

내보내기 절차

1.  왼쪽 **전략 목록**에서 내보낼 전략을 **선택**합니다.
2.  목록에서 **오른쪽 클릭 → 내보내기** 를 선택합니다.
3.  **파일 저장 대화창**이 열립니다.
	*   기본 파일 이름: **{전략 이름}.json**
	*   파일 형식: **JSON 파일 (\*.json)**
4.  저장할 경로와 파일 이름을 선택한 후 **저장** 버튼을 클릭합니다.
5.  성공 시 안내 메시지가 표시됩니다.

매매 전략을 '{저장경로\\파일명.json}'에 저장되었습니다.

내보내기 파일에 포함되는 정보

내보내기 파일(JSON)에는 전략 재현에 필요한 모든 구성 요소가 함께 포함됩니다.

<table><tbody><tr><td><p>포함 항목</p></td><td><p>설명</p></td></tr><tr><td><p><strong>전략 기본 정보</strong></p></td><td><p>이름, 계좌번호, 기본 매수 금액, 보유 종목수 제한, 재매수 제한, 자동 실행 시간 등</p></td></tr><tr><td><p><strong>주문 전략 목록</strong></p></td><td><p>매수/매도 구분, 주문유형, 준비·실행 조건 설정, 금액 계산식 연결 정보 등</p></td></tr><tr><td><p><strong>시스템조건식 목록</strong></p></td><td><p>준비조건·실행조건에 사용된 모든 시스템조건식의 상세 내용</p></td></tr><tr><td><p><strong>키움 조건검색식 이름 목록</strong></p></td><td><p>전략에서 사용된 키움 조건검색식의 이름</p></td></tr><tr><td><p><strong>주문금액 계산식 목록</strong></p></td><td><p>전략에 연결된 금액 계산식 내용</p></td></tr><tr><td><p><strong>거래 제외 기준</strong></p></td><td><p>설정된 매매 제외 카테고리 목록</p></td></tr></tbody></table>

⚠️ 현재 진행 중인 **매매전략 작업(종목별 주문 이력)** 은 내보내기 파일에 포함되지 않습니다.  
또한 내보내기 파일에서 전략의 **실행 상태는 항상 OFF**로 초기화됩니다.

## 5.5.2 전략 가져오기 (Import)

가져오기 전 사전 조건

⚠️ **반드시 확인하세요:**  
가져오려는 전략에서 사용된 **키움 조건검색식 이름**이 내 키움 영웅문 HTS에 **동일한 이름**으로 등록되어 있어야 합니다.  
조건검색식 이름이 없으면 가져오기가 실패합니다.

가져오기 절차

1.  **오른쪽 클릭 → 가져오기** 를 선택합니다.
2.  아래 사전 확인 메시지가 표시됩니다.

외부에서 가져오려는 매매 전략에 사용된 키움 조건검색식이 내 영웅문에 존재해야합니다. (동일한 이름)

계속하시겠습니까?

1.  **예**를 클릭하여 계속합니다.
2.  **파일 열기 대화창**이 열립니다.
	*   **JSON 파일 (\*.json)** 형식의 파일을 선택합니다.
3.  **계좌번호 선택 창**이 열립니다.
	*   이 전략을 적용할 **계좌번호를 선택**합니다.
	*   내보내기 원본과 다른 계좌번호를 선택해도 됩니다.
4.  성공 시 전략이 목록에 추가되고 안내 메시지가 표시됩니다.  
	매매 전략을 '{파일경로}'에서 가져왔습니다.

활용 시나리오

시나리오 1 — 전략 백업

1. 전략 선택 → 내보내기

2. 원하는 경로에 JSON 파일 저장

3. 필요 시 가져오기로 복원

시나리오 2 — 전략 복사 후 변형

1. 원본 전략 내보내기 → 파일 저장

2. 가져오기 → 동일 계좌 선택

→ "동일 이름 전략 존재" 오류 발생

3. 원본 전략 이름 변경 후 재시도

→ 복사된 전략이 추가됨

1.  복사된 전략 수정 후 저장

시나리오 3 — 다른 PC로 이전

\[PC A\]

1. 전략 내보내기 → USB 또는 클라우드에 저장

\[PC B\]

1. 조건검색식 목록 동기화 (9.2 참조)

2. 가져오기 → 파일 선택 → 계좌번호 선택

→ 전략이 그대로 복원됨

주의 사항

*   내보낸 파일은 **JSON 텍스트 파일**이므로 외부에 공유 시 계좌번호 등 민감 정보 노출에 주의하십시오.
*   가져오기 전 해당 전략에서 사용하는 **모든 키움 조건검색식 이름을 미리 확인**하고, 영웅문 HTS에 동일한 이름으로 등록해 두어야 합니다. (→ 9.2 조건검색식 목록 불러오기 참조)
*   동일 이름의 전략은 가져오기가 불가합니다. 기존 전략 이름 변경 또는 삭제 후 재시도하십시오.
*   가져온 전략의 **실행 상태는 항상 OFF**이므로, 자동 매매를 시작하려면 별도로 ON 하여야 합니다.
