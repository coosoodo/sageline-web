※ 회원가입은 하지 않아도 됩니다.

## 3.2 자동 로그인 설정 방법

부엉이 ATS에는 두 가지 독립적인 자동 로그인 기능이 있습니다.

① 부엉이 ATS 자동 로그인 (**IsAutoLogin**)

SageLine 서버 인증(프로그램 라이선스 확인)을 자동으로 처리합니다.

**설정 방법**

1.  로그인 화면 우측 폼에서 **"자동 로그인"** 체크박스를 활성화합니다.
2.  정상적으로 로그인이 완료되면 해당 설정이 **usersettings.json**에 저장됩니다.
