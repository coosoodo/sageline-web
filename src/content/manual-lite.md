# 1. 시작하기 전에

## 1.1 이 문서에 대하여

이 문서는 <strong>부엉이 트레이더 라이트 (Owl Trader Light)</strong>의 공식 사용자 설명서입니다. 라이트 버전은 <strong>키움증권 REST API 기반의 조건검색식 자동매매</strong>에 집중한 경량 프로그램입니다.

부엉이 트레이더(풀 버전)와 달리 AI 분석·차트·서버 연동을 제외하고, 조건검색식으로 매수하고 손절·익절·트레일링 스탑·분할 매매 등 계좌를 지키는 매도 전략에 집중했습니다.

## 1.2 동작 원리

라이트 버전의 자동매매는 다음 흐름으로 동작합니다.

*   키움 HTS(영웅문)에서 미리 만들어 둔 <strong>조건검색식</strong>이 종목을 포착합니다.
*   포착된 종목을 전략에 연결된 <strong>매매 설정 프로필</strong>에 따라 매수합니다.
*   보유 종목은 손절·익절·트레일링 스탑·분할 매도 규칙에 따라 자동으로 청산됩니다.

> ⚠️ 조건검색식은 <strong>키움 HTS에서 먼저 생성</strong>해야 프로그램의 전략 설정 목록에 나타납니다.

# 2. 설치 및 계좌 등록

## 2.1 프로그램 설치

*   설치 파일 <strong>`OwlTraderLight-Setup.exe`</strong>를 실행합니다. .NET Desktop Runtime 10(x64)이 없으면 자동으로 감지하여 함께 설치합니다.
*   <strong>사용권 계약(License)</strong>에 동의하면 설치가 진행됩니다.
*   기본 설치 경로는 `%ProgramFiles%\SageLine\OwlTraderLight`이며, 시작 메뉴와 바탕화면에 바로가기가 생성됩니다.

## 2.2 계좌 등록 (AppKey / SecretKey)

![계좌 AppKey 설정 화면](/images/manual-lite/lite-02.png)

라이트 버전은 별도의 아이디·비밀번호 로그인이 아니라 <strong>계좌별 API 키 등록</strong> 방식으로 인증합니다. <strong>계좌현황</strong> 패널의 API 키 설정에서 등록합니다.

<table><tbody><tr><td><p>입력 항목</p></td><td><p>내용</p></td></tr><tr><td><p><strong>계좌번호</strong></p></td><td><p>8자리 계좌번호</p></td></tr><tr><td><p><strong>계좌명</strong></p></td><td><p>구분하기 쉬운 이름</p></td></tr><tr><td><p><strong>AppKey / SecretKey</strong></p></td><td><p>키움에서 발급받은 REST API 키</p></td></tr><tr><td><p><strong>모의투자 계좌</strong></p></td><td><p>모의투자 환경에서 거래할 경우 체크</p></td></tr><tr><td><p><strong>기본 수동매수 전략</strong></p></td><td><p>HTS 등 외부에서 수동 매수한 종목을 관리할 전략 지정</p></td></tr></tbody></table>

*   <strong>연결 테스트</strong> 버튼으로 키를 실시간 검증할 수 있으며, 저장 시 다시 한 번 검증합니다.


# 3. 시작 가이드 (최초 실행)

프로그램을 처음 실행하면 <strong>시작 가이드(온보딩 위저드)</strong>가 표시됩니다. 다음 3단계를 순서대로 진행하면 자동매매 준비가 완료됩니다.

*   <strong>1단계 — 계좌 등록</strong>: AppKey/SecretKey를 등록합니다. (2장 참조)
*   <strong>2단계 — 매매 설정 프로필 확인</strong>: 매수·매도 규칙이 담긴 프로필을 확인합니다. (6장 참조)
*   <strong>3단계 — 전략 생성</strong>: 계좌·조건검색식·프로필을 연결해 전략을 만듭니다. (5장 참조)

선택적으로 텔레그램 알림, 실행 스케줄, 주문 하드리밋을 함께 설정할 수 있습니다. "다시 표시하지 않음"을 선택하면 다음 실행부터 표시되지 않습니다.


# 4. 메인 화면 둘러보기

메인 화면은 상단 메뉴바, 3열 본문, 하단 로그·상태바로 구성됩니다.

![부엉이 트레이더 라이트 메인 화면](/images/manual-lite/lite-01.png)

## 4.1 계좌현황 패널 (좌측)

총자산(추정 예탁)·주문가능금액·총평가손익·계좌 수익률과 당일 매수/매도 주문 사용량을 보여줍니다. 새로고침 버튼과 API 키 설정 진입 버튼이 있습니다.

## 4.2 매매 전략 목록 패널 (좌측)

등록한 전략이 카드로 표시됩니다. 카드의 <strong>가동/정지 토글</strong>로 개별 전략을 켜고 끌 수 있고, <strong>일괄 실행 / 일괄 중지</strong> 버튼과 <strong>신규 전략 등록</strong> 버튼을 제공합니다.

## 4.3 실시간 보유 종목 감시 (중앙)

보유 종목을 자동/수동으로 구분해 종목명·평균단가·현재가·수익률·평가손익·분할 진행 단계·상태를 실시간으로 보여줍니다. 우클릭 컨텍스트 메뉴로 선택 종목을 일괄 매도하거나 목록에서 제거할 수 있습니다.

![실시간 보유 종목 감시 및 체결 현황](/images/manual-lite/lite-11.png)

## 4.4 실시간 체결 현황 (우측)

시간·계좌·종목·매수/매도 구분·체결 수량/단가/금액을 실시간으로 표시합니다.

## 4.5 업종지수 · 로그 · 실시간 등록 현황

*   우상단에 <strong>KOSPI/KOSDAQ 업종지수</strong>와 등락이 실시간으로 표시됩니다.
*   하단 <strong>로그 패널</strong>(기본 접힘)에 실시간 거래·시스템 이벤트가 기록됩니다.
*   하단 <strong>실시간 등록 현황</strong> 상태바에 계좌별 시세·조건 실시간 등록 건수가 표시되며, 클릭하면 요청 제한 안내 팝업(체결 최대 100건, 조건검색 최대 10건 등)을 볼 수 있습니다.


# 5. 매매 전략 설정

전략은 <strong>"계좌 + 매수 조건검색식 + 매매 설정 프로필"</strong>의 연결입니다. 매매 파라미터 본체는 프로필(6장)에 있고, 전략에서는 어떤 조건식으로 매수하고 어떤 프로필을 적용할지 지정합니다.

## 5.1 전략 만들기

전략 목록 패널의 <strong>신규 전략 등록</strong>에서 전략 설정 창을 엽니다.

*   <strong>적용 계좌</strong>를 선택하면 해당 계좌의 조건검색식이 실시간으로 불러와집니다.
*   <strong>전략 이름</strong>을 입력합니다.
*   <strong>최대 보유 종목 수</strong>를 지정합니다. 이 수를 초과하면 신규 매수가 거절됩니다.
*   <strong>매수 조건검색식</strong>을 선택합니다.
*   적용할 <strong>매매 설정 프로필</strong>을 선택합니다. (필요 시 프로필 관리로 이동)

## 5.2 전략 수정 · 삭제

전략 목록 카드에서 수정하거나 삭제할 수 있습니다.

# 7. 조건검색 신호

## 7.1 조건검색 신호 모니터

조건검색식·종목별 당일 수신 건수와 포착·편입·이탈 신호를 실시간으로 보여줍니다. 신호 이력(검색 종류, 매수/매도 구분, 생성 시각)도 확인할 수 있습니다.

## 7.2 가상 신호 생성기 (테스트용)

실거래 없이 자동매매 로직을 검증하는 테스트 도구입니다. <strong>편입/이탈 신호</strong>(계좌·전략·종목 지정)와 <strong>포착 신호(다중)</strong>(종목코드 목록)를 임의로 발생시켜 전략이 정상 동작하는지 확인할 수 있습니다.


# 8. 운영 도구

<strong>설정</strong> 메뉴에서 다음 도구에 진입합니다.

## 8.1 시스템 설정

디자인 테마(다크/라이트, 변경 시 재시작), 계좌평가잔고 API 요청 주기(권장 최소 10초), 하이브리드 모드(자동·수동 혼용), 실시간 조건 판단 로그 기록, 프리·애프터마켓 지정가 변환, 이상 시세 가드 등을 설정합니다.

특히 <strong>주문 폭주 방지 하드리밋</strong>으로 일일 매수/매도 한도, 종목별 한도, 1회·일일 금액 한도, 반복 주문 한도를 지정해 오작동으로 인한 대량 주문을 막을 수 있습니다.

## 8.2 매매 통계 · 성과 분석

![매매 통계 및 성과 분석 화면](/images/manual-lite/lite-07.png)

계좌·전략·기간으로 필터하여 청산 거래 수·승률·총 실현손익(추정)·평균 수익/손실·손익비(Profit Factor)·MDD·최대 수익/손실을 카드로 보여줍니다. 전략별/종목별/거래 내역 탭을 제공합니다.

## 8.3 보유종목 일괄 매도

주문 종류·가격 구분·호가 조정을 지정하고, 종목을 미리 보고 선택해 한 번에 매도합니다.

## 8.4 프로그램 실행 스케줄

![프로그램 실행 스케줄 화면](/images/manual-lite/lite-09.png)

자동 실행 시작/종료 시간과 요일을 지정합니다(07:30~23:00 범위). Windows 작업 스케줄러를 사용하므로 등록 시 관리자 권한이 필요하며, 종료 시각에는 프로그램이 스스로 정상 종료합니다.

## 8.5 설정 내보내기 / 가져오기

전략·프로필 등의 설정을 파일로 내보내고 가져올 수 있습니다.

## 8.6 로그 파일 관리자

![로그 파일 관리 화면](/images/manual-lite/lite-10.png)

로그 보관 기간을 설정하고, 파일 목록(크기·수정일)을 확인하거나 메모장으로 열고 삭제할 수 있습니다. 문제 진단이 필요할 때 <strong>로그 파일을 서버로 전송</strong>할 수 있습니다.

> 라이트 버전은 매매·시세용 외부 서버와 연동하지 않으며, 이 기능은 진단 목적의 로그 업로드에 한정됩니다.


# 9. 텔레그램 알림

라이트 버전은 매매 알림을 <strong>텔레그램</strong>으로 전송합니다. (Discord는 지원하지 않습니다.) <strong>설정 → 텔레그램 알림 설정</strong>에서 구성합니다.

![텔레그램 알림 설정 화면](/images/manual-lite/lite-08.png)

*   Bot Token은 텔레그램에서 <strong>@BotFather</strong>로 봇을 만들면 발급되고, Chat ID는 <strong>@userinfobot</strong>으로 확인할 수 있습니다.
*   <strong>기본 테스트 전송</strong> 버튼으로 연결을 확인합니다.


# 10. 부록

## 10.1 실시간 등록 한도

키움 API 제한에 따라 계좌별로 실시간 등록 건수가 제한됩니다. 하단 상태바의 실시간 등록 현황을 클릭하면 현재 사용량과 한도(체결 데이터 최대 100건, 조건검색 최대 10건 등)를 여유·주의·포화 색상으로 확인할 수 있습니다.

## 10.2 키움 접속 단절 시간

월~토 07:50~07:55, 일 05:00~05:15에는 키움 서버 접속이 일시적으로 끊깁니다. 이 시간에는 주문·조회가 제한됩니다.

## 10.3 버전 정보
